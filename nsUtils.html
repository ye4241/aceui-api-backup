ping某地址，看是否通

编号  :  000003

void - NSISPingSomeWWW
请求参数
参数名 类型 必传 描述
wwwAddress string Y 服务器地址 例如 www.baidu.com 或 ip
tempFilePath string Y 结果临时文件存放地址
使用举例
Function .onInit
    GetFunctionAddress $0 pingSomeWWW
    BgWorker::CallAndWait
FunctionEnd

Function pingSomeWWW
    nsUtils::NSISPingSomeWWW "www.baidu.com" "$TEMPfuck.txt"
    Pop $1
    MessageBox MB_OK "$1"
FunctionEnd
备注
因为处理中含有 sleep（1000）；
所有需要再线程中处理
